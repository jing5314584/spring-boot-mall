上海启旺健康管理有限公司获得A轮融资

时间：2016-02-01 00:00:00 来源：未知 点击次数：0次

上海启旺健康管理有限公司（简称：启旺）旗下APP“一乎百医”2015年4月一经上线便得到多方投资人的关注，2016年2月获得泰然金融集团3000万A轮投资，这是启旺一个新的起点

泰然金融集团（简称：泰然金融）是一家专注于为中国中小企业及个人提供全方位金融服务的新兴多元化互联网金融企业。2012年于杭州成立，凭借雄厚的资金实力和专业的管理经验，2014年正式变更为集团公司，累计注册资本5.8亿元，同时总部迁移至上海。集合并融合有投资理财、资产管理、第三方产品、互联网金融、众筹、大数据、云储存、金融搜索、金融社交、私募股权、项目融资等专业服务和业务模块，提供全面的、安全的、可信的、绿色的财富增值服务。

启旺和泰然一致达成“乎之间，百医相伴”的服务理念。获得注资后，资金主要用于扩展医疗资源、产品优化、用户推广、用户服务，为客户带来更多专业医疗知识和提供科学的健康指导。
